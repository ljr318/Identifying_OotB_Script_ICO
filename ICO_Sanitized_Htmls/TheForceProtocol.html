Technical features

To solve the problems in the development of the DApp for Ethernet smart contracts are not easy to upgrade, data structure solidification, slow interaction on the chain, poor user experience, lack of necessary infrastructure, security issues are prominent

Foundation components

APEC

APEC, as the core On-Chain architecture, is based on Solidity smart contracts, and has adapted and optimized the inconveniences of contract development while adhering to the premise of decentralization and asset ownership.

Extension components

BEAMS

BEAMS is an Off-Chain system that works closely with the contract, continuously monitoring the operation of the contract, auditing data and assets, speeding up the product's responsiveness, smoothing out the fluctuating curve .

Financial components

GEL&CALM&MAK

The financial component includes three parts: Global Emergency Lockdown(GEL), Cooperative Automatic Lockdown Mechanism(CALM), and Multisig Admin Keys(MAK). Secure the platform from attacks and intrusions.
