What is SWIPE?

Subscribe for SWIPE updates

Problem: Trust Deficit in Data Monetization

User data is scattered across different apps.
Today's centralized data intermediary approach is not transparent and at times unfair.

SWIPE Network - a Blockchain Infrastructure

SWIPE Network is a decentralized mobile engagement data platform/infrastructure that empowers app developers with a suite of marketing tools and SDKs (software development kit) to improve app engagement and user retention, while allowing for transparent and fair data monetization.

Enabling Privacy and Data Sharing Protocol

Giving control of data back into the hands of the users

User Consent

Opt in/out of data monetization campaigns. Full control of privacy and data sharing policies

End-to-End Encryption

Secure data management

Smart Contract

Data requests and reward distribution are managed and logged transparently

Data Re-Encryption

Data is re-encrypted for requestor's access when data sharing permission is given.
