WandX Basket Protocol
Ethereum, NEO, Wanchain and Aion

Create, buy and trade a group of theme-based ERC20, NEP5 or WRC20 tokens created
using the WandX Basket Protocol, a proprietary risk-hedging product.

NEP5 based Token Baskets

Create custom Token Baskets using 2 or more NEP5 Tokens, deposited into the smart contract to be assigned to a user

ERC20 based Token Baskets

Create a custom smart contract Token Basket using 2 or more ERC20 Tokens, to be assigned to any wallet address that transfers the ask price worth of ETH to the Basket address. View in Exchange

WRC20 based Token Baskets

Create a custom token basket of 2 or more WRC20 tokens, deposited into the smart contract to be assigned to a user

ATS based Token Baskets

Create a custom token basket of 2 or more ATS (AION Token Standard) tokens, deposited into the smart contract to be assigned to a user
