What is Walleteum?

The Walleteum is a wallet management system that allows users quickly and easily interact with their coins, while tracking and monitoring diverse assets. Walleteum will allow users to view, send and receive balances of any supported cryptocurrency.


The Walleteum offers users a variety of features not afforded by other wallets, leveraging the unique capabilities of the Walleteum platform. Mobile support is a priority for Walleteum so that consumers can have access to their coins wherever they go.

  • Which programming languages are used in Walleteum?

    Walleteum is written in C# and Go. Front-end applications are built with React, Swift for iOS, and Java for Android.

  • Does Walleteum exchange come with an example trading client?

    Yes. The example allows a trader to login with a username and password, allows the account info to be updated for testing purposes (buying power, cash, etc.), and allows the tester to place trades.

  • Which third party libraries are used in Walleteum?

    NLog, log4net, Newtonsoft JSON, and SuperWebSocket.

  • Where does MyExchange store data?

    All account, order and market price data will be stored on servers. There are no connections to Modulus servers.
