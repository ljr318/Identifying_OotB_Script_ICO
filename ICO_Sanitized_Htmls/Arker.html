What is ARKER?

ARKER is a decentralized cryptocurrency created with the purpose of rewarding the player.

Each time a player completes a milestone can be rewarded with an ARKER amount. The player can use this token to acquire items in the game, exchange it or even sell it to get money. All this in order to create an in-game economy.

What is Arker: The legend of Ohm?

Is the first RPG online videogame that rewards the user with a cryptocurrency.

The first videogame using ARKER to reward the player for each battle they win. The whole in-game economy of the game is based on this token: from founding a clan up to buying and selling items in the market is made using ARKER.
