MINING POOL Guide

Follow the ESN Pool Status link below and choose a pool that you want to use.

A. https://escpoolstats.gonsmine.com

Download Claymore

A. Visit the link below https://bitcointalk.org/index.php?topic=1433925.0
B. Click the link next to 'MEGA'
C. Select the file depends on OS you use > right click > Download > Standard Download
D. Begin download at the bottom left
E. Right click > open folder

Decompress

A. Decompress the file downloaded
B. After decompression, copy the folder
C. Paste the folder at the wallpaper of your PC

Edit 'Start.bat'

A. Double click 'Claymore'sâ¦.' folder
B. Select 'Start.bat' > right click > edit
C. There could be various commands on note. Delete all the commands.
D. Copy the command below (you can see the command on the mining pool web)
E. Paste the command at 'start.bat'
<Pool Info> : Pool information which you chose (Ex: esn.gonsmine.com:8008)
<address> : Your ESN wallet address
<worker> : mining worker name(If you have just one, you can delete '-eworker <WORKER>')
Worker means each mining machine. Several mining machines can be connected to one wallet.
F. Type your ESN wallet address
ex) My ESN wallet address : 0x3409025dCE86Ad441A5a80f30CE03768d37E40Bc
The name of my mining machine : rig1

EthDcrMiner64.exe -epool pool-asia2.ethersocial.org:8008 -esm 0
-ewal 0x3409025dCE86Ad441A5a80f30CE03768d37E40Bc -eworker rig1 -allcoins 1 -allpools 1
G. Save and close the note
H. Run 'start.bat'
It could not work because of some antivirus programs.
In this case, modifies some options on the programs to stop realize mining program as a virus.
I. After run 'start.bat', mining begins.

Check mining

A. Visit ESN mining pool web which you use
B. Type your Wallet address in 'Your Stats & Payment History'
C. Click 'Lookup' to check your mining information

1. If both Pending Balance and Total paid don't show up

In this case, mining is not working well. There should be some issues. Double check your 'start.bat' and wallet address.

2. Pending Balance shows up, but Total paid doESN't show up

It means that mining is working well, but it is not paid yet. It usually take some time to pay depend on hash rate. Therefore, you can get paid after few hours in general. However, ask to pool manager for this issue if it is not paid after few hours. There could be some issues on the mining pool.

3. Both Pending Balance and Total paid show up, but no balance on your wallet

There is no issue on your mining. Check version of your wallet or synchronize the wallet
