General Questions

What is cryptocurrency?

Cryptocurrencies are monetary units (also known as "coins" or "tokens") protected by cryptographic technology. There is no physical analogue to these monetary units, they exist only in virtual space. Tokens are protected from forgery, because they represent encrypted information that cannot be copied (the use of cryptography defined the prefix "crypto" in the title).
You can read more here on Wikipedia's Cryptocurrency Page .

What is Blockchain?

Blockchain is a distributed database that doesnât have storage devices connected to a shared server. This database stores an ever-growing list of ordered records, called blocks. Each block contains a timestamp and a link to the previous block.
Read more here on Wikipedia's Blockchain Page .

What are Smart Contracts?

A smart contract is a computer protocol intended to facilitate, verify, or enforce the negotiation or performance of a contract . In other words, smart contracts are program code that allow you to automatically perform certain functions when certain conditions occur. To implement smart-contracts, a decentralized environment is required that completely excludes the human factor. To use the transfer of value in a smart-contract, it requires a cryptocurrency.
Get more details at Wikipedia's Smart Contract Page .

What is the Long Term Bounty Fund?

The Long-term Bounty Fund will be used for Eloplay Smart Tournaments and for the Skill-Based Token Distribution System.
