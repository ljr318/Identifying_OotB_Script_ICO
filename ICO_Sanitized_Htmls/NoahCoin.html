ノアシティ について

ノアシティ は 、 2019 年 11月 2 日 に 設立 さ れ まし た 。 公式 の 設立 記念 日 は 、 東京 で 開催 さ れ た ノアホルダー 向け の 最初 の ミートアップ の 日付 と 一致 し て い ます 。

ノアシティ2.0 の 政治 組織 は 直接 民主 主義 です 。 主 な 政府 機関 は 市民 によって 管理 さ れる 司法 省 、 外務省 、 財務省 そし て 日本 語訳 手伝い に なる 文科省 です 。

すべて の プロセス は 、 デジタル 投票 によって 規制 さ れ て い ます 。 DPoS に 基づい た 有望 な テクノロジー は 完全 な 透明 性 を 付与 し 、 第 三 者 の 関わり を 除去 し ます 。 国 の 支払 い 手段 として は 、 NOAH コイン に が 使い ます 。

国民 が 国 に お金 を 収める の で は なく 、 国 が 国民 に お金 を 公平 に 分配 し ます 。 利益 分配 は 、 2020 年1月 に 開始 さ れ 、 獲得 できる 予想 の 不労 取得 は 、 年間 100 ％ まで です ！
