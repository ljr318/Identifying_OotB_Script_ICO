Few Steps For a Crypto To Cash

Scroll Down

  • 01. User Visit Our Website / Apps.

    User select which crypto currency they want to exchange
    from list of available currencies.

  • 02. Location Selection.

    The user can select Country & city from the list
    where they want to receive the cash.

  • 03. Enter The Amounts

    Our system checks available cash limits from agent for the selected city.
    The user can enter the amount they want to exchange
    between minimum & maximum limits.

  • 04. Send the Crypto Currency

    Now user can send the deposit amount to displayed address and wait
    for confirmation on the blockchain network,
    and as soon as deposit confirms an eight digits unique pin will generate and shown on
    the user screen, and notification sends to the user via email and SMS.

  • 05. Visit our Nearby Location.

    Now user can visit there nearest agent/cash point location & provide transaction no & 8 Digit Pin.
    Our agent checks the details from our apps & request for OTP.

  • 06. Receive the Cash.

    The user gives otp to our agent & agent verifies it in our apps & if otp verification successes
    our agent provides cash to user & transaction is completed.
