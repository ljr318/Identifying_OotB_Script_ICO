Overview
Technology Introduction
CONUN的分布式计算(DistributedComputing)提供了一个通用计算网络架构平台，支持基于桌面网格计算技术的连接到互联网的个人计算机的分布式处理。

CONUN由分布式计算资源共享参与者和用户之间的合约运营，并为所有参与者提供开放和公平的利润生态系统。
Architecture
CONUN设计一个计算节点，用于控制个人计算机的组件，如CPU，GPU，内存和存储设备，模块化每个节点以及处理应用程序。
Grid Computing Core
使用计算资源来解决计算任务的分布式计算核心功能
Virtual Machine(Docker Container)
可在每个计算节点上部署并自动运行应用程序，而不受各个操作系统干扰的独立虚拟处理系统
Hybrid Peer-to-Peer Network
不区分计算机的类型，充当网络上的所有计算机客户端和服务器
Ethereum smart contracts(交易系统)
透明而安全地交易，合约并付款而不需中介服务的系统
Use Cases
科学计算系统
可用于执行需要复杂和快速计算性能的科学计算任务。
深度学习模型项目
在人工智能领域中的卷积神经网络（CNN），循环神经网络（RNN）和强化学习等另一种可通过用户设计的深度学习模型，快速、深度学习算法，造成多样的程序环境。
大数据分析项目
应用大数据分析技术处理数据挖掘、自然语言处理、模式识别处理等基本方法，处理社交媒体等它应用大数据分析技术处理数据挖掘、自然语言处理、模式识别处理等基本方法，处理社交媒体等文本挖掘、意见挖掘、交网络分析和聚类分析。
计算机图形处理项目
计算机图形学领域的计算机生成图像（CGI）被用于快速且低成本地处理电影、电视节目、广告、模拟器、模拟和3D计算机图形等项目。
生命科学研究项目
改善现有治疗疾病和开发新疗法的生物学研究是利用分布式计算的领域之一，可用于大学研究机构和生物技术行业的研究。
学术/研究/社会文化热门等服务
通过直接或间接参与各种学术研究，社会文化话题等通用网络获得额外利润，同时创造一个通过研究和讨论提高生活质量和发展文化的社区，可以创造各种学术研究，社会和文化问题。
