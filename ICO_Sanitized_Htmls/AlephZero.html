Meet Aleph Zero

Public blockchain with private smart contracts

Aleph Zero is a peer-reviewed DAG platform that’s scalable, fast, and developer-friendly.

It’s a Proof-of-Stake public blockchain with private smart contracts built by a world-class team that aims to reach mass adoption of Distributed Ledger Technology.

why it works

SCALABILITY

Aleph Zero achieves 100,000 TPS in a fully decentralized system. We’re running the tests with Amazon Web Services on 128 nodes across 5 continents.

DECENTRALIZED

Aleph Zero is a first decentralized Directed Acyclic Graph (DAG) with a large, rotating, random committee.

PRIVATE SMART CONTRACTS

Customize functionality for automation and programmable, private contracts.

FREE VALUE TRANSFER TRANSACTIONS

Simple value transfers are, and always will be, free.

Aleph Zero Ecosystem

Building a new world

Aleph Zero ecosystem consists not only of the consensus but is also capable of synchronizing with other blockchains and running smart contracts, as well as a truly trustless universal wallet — which we call Common.

Aleph Zero is an ecosystem that consists of:

  • Consensus

    A peer-reviewed, DAG-based protocol

  • Cloud

    Decentralized file storage / IPFS

  • Private Smart Contracts

    Scalable, self-executing, private smart contracts

  • Common

    Decentralized Exchange (DEX) and a Decentralized Dark Pool

  • Oracle

    Information bridge between the external world and the Aleph Zero ecosystem

Why we are building Aleph Zero

To solve unsolvable

  • The Security Challenge

    Distributed Ledger Technologies are considered immune to any attacks thanks to their decentralization. But in many cases, they don’t provide theoretical security guarantees which makes them less practical to use long-term.

    Aleph Zero provides both practical and theoretical security guarantees.

  • The Scalability Challenge

    As known research points, there is a trade-off between security and speed in every blockchain. When operating on a high scale, to maintain security, most blockchains are bound to low transaction speeds.

    Aleph Zero is both secure and blazingly fast when operating on a high scale.

  • The Cost Challenge

    Blockchains that are based on Proof-of-Work protocols are highly inefficient at scale. Mining requires high computing power which contributes to high energy consumption. This is only to run the network.

    Aleph Zero is mining-free.

  • The Speed Challenge

    The point of Proof-of-Work protocol is to slow the network down. Many blockchains, in order to improve speed and scalability, are sacrificing the decentralization and security. You either send your assets securely and wait for even 40 minutes or risk.

    Aleph Zero performs transactions in seconds.
