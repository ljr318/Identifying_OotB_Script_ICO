Why Us?

GECKO Crypto1 is a compliance system that enables a self-regulatory framework on behalf of the Token Sale market.

Most importantly, it provides a verifiable independent blockchain audit trail of compliance to satisfy global financial regulators. GECKO Crypto1 is implemented on the blockchain, across an entire ecosystem of data providers and consumers.

The Blockchain is a distributed ledger technology on which transactions are made and recorded both chronologically and publicly. With a compliance ecosystem, we can enable a self-regulatory function on behalf of the Token Sale market, enabling better transparency, compliance & accountability to investors.

All data is live, immutable, auditable and verifiable.

  • GECKO Governance is a multi-award winning Regtech solution for managing financial services compliance.
  • The Company has a global team of staff based in New York, Ireland & Australia.

Benefits of Token:

The GECKO Token (GECKO) is a utility token that can be redeemed to access the existing suite of GECKO products and services. Token Sale Issuers, Investors, and Developers can use GECKO tokens to access, utilize and provide input to the GECKO network. GECKO tokens allow value created in the system to be captured by the system itself.

Enabling Better Investor Transparency, Compliance & Accountability for the Token Sale Ecosystem.

Certification

A “Stamp of Attestation” will provide Token Issuers with industry validation and accountability to investors that the GECKO “best practice” framework of Token Sale governance is being utilized.

Transparency

The GECKO Crypto1 system will provide real-time transparency at each stage of the Token Sale project through the use of its dashboard, documentation, project management and blockchain integration.

Compliance

The GECKO Crypto1 system seeks to becomes the industry standard for “Best Practice” Token Sale governance and compliance. The system will provide permissioned regulator access.

Accountability

The GECKO Crypto1 system will provide detailed accountability on all Token Sale projects. This functionality will include initial and ongoing project milestone updates, distribution and collaboration of all Token Sale documentation and real time oversight reporting for investors
and regulators.

How it Works

At its core, the GECKO Crypto1 system is a collection of processes for managing the complex interactions between participants in the Token Sale ecosystem.
