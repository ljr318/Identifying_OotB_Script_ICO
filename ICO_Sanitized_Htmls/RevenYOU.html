Veelgestelde vragen

Wat is een bot?

Een bot is een computerprogramma dat is ontworpen door handelaren of ontwikkelaars, die op de juiste momenten voor jou activa inkoopt en verkoopt op basis van een handelsstrategie. Een bot wordt ook wel een algoritme genoemd.

Hoe start ik een bot?

Om een bot te starten, druk je op het knopje bots onderin je scherm. Vervolgens kies je een bot uit die jou aanspreekt. Druk op de knop start this bot en selecteer het bedrag dat je wilt storten. Om een storting te doen moet je een account aanmaken.

Hoe stop ik een bot?

Op je dashboard staat je actieve bot. Druk hier op en druk op stop this bot. Je geld staat nu in available funds en kan je terug laten storten of herinvesteren in een andere bot.

Rekenen jullie kosten voor het terug laten storten?

Ja, we rekenen 1,5% uitstapkosten wanneer je je bedrag terugstort vanuit je available funds naar je bankrekening. Je kunt ook het geld in je available funds laten staan als je nog niet wilt herinvesteren in een bot, dit scheelt transferkosten.

Kan ik extra geld op een bot plaatsen?

Nadat een bepaald bedrag aan een bot toegewezen is, kan dit bedrag niet worden verhoogd. Wat je wel kan doen is de bot stoppen en dezelfde strategie opnieuw kiezen met een nieuw bedrag.

Rekenen jullie kosten voor het storten van geld op een bot?

We rekenen 0,2% instapkosten wanneer je een bedrag stort naar onze BOTS platform. Wanneer een bedrag in available funds zit en je wilt vanuit daar een bot activeren, kost dit 0,2% van het bedrag dat je inlegt.

Worden mijn gelden veilig opgeslagen bij BOTS?

BOTS biedt gebruikers een van de veiligste handelsomgevingen, dankzij onze talrijke relaties met gevestigde partijen zoals de topbeurzen en wallet leveranciers.
