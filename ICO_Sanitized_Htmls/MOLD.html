About us

What is Cosmos Gaming Hub?

Cosmos Gaming Hub is a fair and secure distributed game platform that supports the development of new games and makes it easier to trade valuable data such as items, weapons and armor in the game. Items, weapons, armors etc in the games will be distributed to the market as uniquely defined tokens on the Cosmos Gaming Hub chain, enabling transactions that do not require third parties. Unlike the conventional centralized game structure, Cosmos Gaming Hub creates new value for data in a game virtual space because token ownership does not belong to the game operator. The new value created in virtual space has the potential to start to have more realistic value in the future, and to create a larger economy.

Cosmos Gaming Hub Architecture

Ordinal blockchain games build on Ethereum are not enough to achieve good User Experience. Cosmos Gaming Hub will build own PoS based blockchain for gaming platform applications using tendermint. Realizing high thoroughput blockchain for games will significantly improve usabirity.

And implementing ABCI(Application BlockChain Interface) allow Cosmos Gaming Hub to connect cosmos-hub which will bring various economic benefits to users in Cosmos Gaming Hub.

Cosmos Gaming Hub Features

Flexibility

Any technical updates(including changing validator numbers etc) and early bug fixs are possible with a voting system in our ecosystem.

Interoperability

All assets and tokens on the Cosmos Gaming Hub chain will be made inter-operable by IBC(Inter-Blockchain Communication) protocol in the near future.

Gaming SDK

Game developers can easily build their games on Cosmos Gaming Hub using Cosmos Gaming Hub SDK which will help connect to our network.

Secure

The Cosmos Gaming Hub chain operators are themselves Stakers and Delegators in a BFT Proof-of-Stake system in the Cosmos Gaming Hub.

Scalabirity

Scalability is realized as a Layer 1 solution by using tendermint-BFT based PoS consensus algorithm in Cosmos Gaming Hub main chain.

High Throughput

Based on PoS consensus algorithm, ten thousands of transaction per second will be able to reach in the future.

Project Start

Project MOLD(now re-branded as Cosmos Gaming Hub) started. Previously we planned to build gaming platform on Ethereum.

Original chain Idea Release

Released MOLD design document. At this time we changed to build own blockchain for gaming platform.

2019

Testnet Release

PVM tendermit-Based Blockchain will be released in addition to the Cosmos Gaming Hub Chain concept paper.

2020

Mainnet Launch

After the several monthes stable operation, We also plan to accomplish the connection with Cosmos Hub in mainnet.

2021

We will have a beta version of the Cosmos Gaming Hub SDK ready for game developers so that they can start building on our network.
