What is Fysical?
Infrastructure

Fysical is the infrastructure for the transparent and compliant exchange of location data; foot traffic sensor readings, store visit information, commute routes.

Data Buyers & Suppliers

Data suppliers can share and monetize their location data, maintaining transparency, ownership, and control. Data buyers can access and validate data and its source.

Transparency

Fysical directly contrasts with today's closed systems, data brokers, and black boxes, and replaces them with a transparent peer-to-peer blockchain protocol.

Fysical Protocol

Data Suppliers contribute data to FysicalÂ

Anyone can supply data to Fysical. Third-parties also provide software tools for data suppliers, such as a mobile SDK, that make it easy to securely share data to the blockchain.

Verifiers and Enrichers

Verifiers validate and score data for accuracy (was this datapoint a Starbucks visit, or was it actually the Burger King next door?). Enrichers process raw data into a more valuable dataset. For example, Enrichers map GPS latitude and longitude values to a Point-of-Interest, indicating a visit to that Point-of-Interest. Enriched data creates many more use cases.

Marketplaces

Marketplaces are built on top of Fysical to provide businesses, machines, and individuals with an easier way to search for and buy data. Marketplaces can curate valuable datasets and keep track of data supplier and data buyer reputations.

Data Buyers can search for and license data

Any business, individual, or machine can freely query Fysical to buy data. Data buyers can see an audit trail of all data contributed to the blockchain that clearly identifies the source of the data. Third-parties supply tools for data buyers, like Marketplaces, that make it easy to search for and buy data.

Community

Consumers claim and authorize the use of data and provide crowdsourced data that is used by Verifiers for data accuracy verification, training data for machine learning, and to improve point-of-interest data for Enrichers.

All market participants are paid in FYS tokens

Data Suppliers, Marketplaces, Verifiers, Enrichers, and Consumers are paid
