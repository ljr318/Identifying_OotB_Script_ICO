Big Idea

To erase the border between game currency and real currency

Competitive Advantages

Problems of existing games

  • The currency is isolated, it can only be used within a game itself
  • Lack of value in the real world
  • A valuable reward for professional cyber-sportsman only

CryptoSoul Model

  • A unified game currency for all games
  • SOUL token can be exchanged for Bitcoin or Ethereum
  • Each player receives SOUL for his game achievements

SOUL Economics

Burn
  • 20% of SOUL spent in games we redeem on the exchange
  • We burn purchased tokens
Mint
  • Tokens are issued only to collect investments for a new game
  • Tokens' issue is limited to 1.37 million SOUL per day
The average player reward decreases as the SOUL price rises. This provides inflation control
