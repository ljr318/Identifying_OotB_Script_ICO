Roadmap

2013
  • Playkey video streaming development begins
2015
  • The cloud-based Playkey.net gaming service launches
2016
  • Playkey.net expanded in Europe and UK with 2.5M users
2017
  • ICO Pre-Sale
  • PKT ICO and Blockchain Integration
  • The first cryptocurrency exchanges are entered
  • Planning the architecture of a decentralized system
2018 Q1
  • Starting to develop a decentralized system: the first architectural components, code-review of the current structure with an eye toward future decentralization
  • Support for videocards based on Nvidia GTX
2018 Q2
  • Developing a decentralized system: host (server) control elements, the first testing client on Playkey’s own servers
  • Prepping for the launch of a centralized platform in South Korea and the USA
  • Adding payment in tokens for subscription to the current centralized Playkey.net platform
2018 Q3
  • Developing a decentralized system: testing miner support in the ecosystem (alfa-version)
  • Implementation of Desktop technology needed for decentralized architecture
  • Transfer of centralized version of the platform to Desktop
2018 Q4
  • Development of technology to stabilize streaming between miners and gamers
  • Development of quality control system for gaming sessions to enable payments between miners and gamers
2019 Q1
  • Developing a new method for capturing images
  • Developing a new way of displaying cursors
2019 Q2
  • Scaling in Europe, increasing the number of paid users in Europe alone to 30,000 a month
  • Increasing user playtime to 400 min/month
2019 Q3
  • Improving the alfa-version of a decentralized system, launching personal mining accounts, inclusion of gamers and miners in the ecosystem (beta-version)
2019 Q4
  • Expanding the network of beta-testing miners
  • Debugging the update and server-search systems so that servers are selected with the geographic area in mind
2020 Q1
  • Automating the connection process for miners
  • Implementing a billing system for miner accounts
2020 Q2
  • Expanding the miner network in Russia and other CIS countries
  • Testing the decentralized model in North America, Europe, and Asia
  • Running tests for the billing system (with the ability for miners to withdraw earnings)
2020 Q3
  • Migrating the Playkey service in use to the decentralized server base (for phase 1, the server ratio is 20:80, with 20 being the decentralized servers)
  • Testing the decentralized model with the ability for users to select servers manually
  • Expanding the miner network in Europe
2020 Q4
  • Migrating the Playkey service in use to the decentralized server base (for phase 2, the server ratio is 40:60, with 40 being the decentralized servers)
  • Expanding the miner network in South Korea and the US
